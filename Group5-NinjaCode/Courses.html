Course 1 - Introduction to Coding for Kids
Rating: 4.5/5
This course introduces kids to the basics of coding using visual programming languages. Students will learn fundamental concepts such as loops, conditionals, and variables through engaging activities and projects.
Prerequisites: None
Schedule: Monday 9:00 AM - 11:00 AM
Current Tutor: Norah Alghamdi
Expected Objective: By the end of the course, students will be able to understand and write simple programs using visual programming languages.
Course 2 - Building Interactive Websites for Kids
Rating: 4/5
In this course, kids will learn HTML, CSS, and JavaScript to create interactive websites. They will explore various web design principles and learn how to incorporate multimedia elements into their projects.
Prerequisites: Basic understanding of coding concepts from Course 1 or equivalent experience.
Schedule: Tuesday 2:00 PM - 4:00 PM
Current Tutor: Mohammed Alothman
Expected Objective: By the end of the course, students will be able to design and develop their own interactive websites using HTML, CSS, and JavaScript.
Course 3 - Introduction to Game Development for Kids
Rating: 4.8/5
This course introduces kids to the world of game development. They will learn how to use game development tools like Scratch or Unity to create their own games. Students will explore game design principles and learn basic programming concepts specific to game development.
Prerequisites:Basic understanding of coding concepts from experience.
Schedule: Mondays 6:00 PM - 4:00 PM
Current Tutor: Sarah Albadr
Expected Objective:By the end of the course, students will be able to design and develop their own simple games using game development tools like Scratch or Unity.
Course 4 - Advanced Coding Concepts for Kids
Rating: 4.7/5
This course is designed for kids who have a solid foundation in coding and want to explore more advanced concepts. Students will dive deeper into topics such as data structures, algorithms, and object-oriented programming through hands-on projects.
Prerequisites: Basic understanding of coding concepts from Course 1 or equivalent experience.
Schedule: Tuesday and Thursdays 4:30 PM - 6:00 PM
Current Tutor: Mishal Alotaibi

Course 5 - Introduction to Robotics and Coding for Kids
Rating: 4.9/5
This course combines robotics and coding to introduce kids to the exciting field of robotics. Students will learn how to program robots using block-based coding languages and explore various sensors and actuators.

Schedule: Sundays, 5:00 PM - 7:00 PM
Current Tutor: Yasmin Ahmed
